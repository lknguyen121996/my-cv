## 💻 About This Repository

This repository contains the source code for my web-based CV. It was built using:
* **HTML5:** Semantic structure for better accessibility.
* **CSS3:** Custom variables, clean layout, and optimized for `@media print` (meaning you can hit `Ctrl + P` to generate a perfect PDF version directly from the browser).
* **Zero Dependencies:** No heavy frameworks, just pure, fast-loading code.
